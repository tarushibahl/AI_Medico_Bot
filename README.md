# 🩺 AI Medico Bot

An AI-powered medical report analysis chatbot built using **Streamlit**, **LangChain**, **Google Gemini**, **Ollama**, **Hugging Face Embeddings**, **FAISS**, and **ChromaDB**.

The application enables users to upload medical reports in PDF format, build a vector database, generate summaries, explain medical terms, and ask questions about their reports using Retrieval-Augmented Generation (RAG).

> **Disclaimer:** This application is intended for educational purposes only and should not be used as a substitute for professional medical advice, diagnosis, or treatment.

---

# ✨ Features

- 📄 Upload medical report PDFs
- 📝 Generate concise report summaries
- 🤖 Ask questions about medical reports
- 🩸 Explain medical terms (RBC, WBC, Hemoglobin, etc.)
- 🔍 Retrieval-Augmented Generation (RAG)
- 🧠 Supports **Google Gemini** and **Ollama**
- 📚 Supports **FAISS** and **ChromaDB**
- 🎯 Multiple embedding model options
- 🌐 Interactive Streamlit interface

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/tarushibahl/AI_Medico_Bot.git

cd AI_Medico_Bot
```

---

## 2. Create Virtual Environment

### Windows

```cmd
python -m venv medico_env
```

Activate

```cmd
medico_env\Scripts\activate
```

### macOS / Linux

```bash
python3 -m venv medico_env
```

Activate

```bash
source medico_env/bin/activate
```

---

## 3. Install Dependencies

Windows

```cmd
pip install -r requirements.txt
```

macOS / Linux

```bash
pip install -r requirements.txt
```

---

# 🔑 Google Gemini Setup

Create a `.env` file in the project root.

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

Generate your API key from:

https://aistudio.google.com/app/api-keys

# 🦙 Ollama Setup (Optional)

## Install Ollama

### macOS

```bash
brew install ollama
```

Or download it from:

https://ollama.com/download

### Verify Installation

```bash
ollama --version
```

### Start Ollama

```bash
ollama serve
```

Open another terminal window.

### Download Model

```bash
ollama pull llama3.2
```

Verify:

```bash
ollama list
```

# 🔄 Application Workflow

1. Upload Medical Report
2. Load PDF
3. Split into text chunks
4. Generate embeddings
5. Build FAISS or Chroma vector database
6. Retrieve relevant context
7. Generate answers using Gemini or Ollama
